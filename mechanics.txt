# 🧱 PHYSICS SYSTEM (SIMPLIFIED BUT SMART)

This is NOT heavy physics (no rigid body engine), but **simulation-based physics**.

## 📦 Collision Types

| Type       | Behavior                   |
| ---------- | -------------------------- |
| Solid      | Cannot pass (walls, trees) |
| Soft       | Slows (grass, crops)       |
| Liquid     | Movement penalty (water)   |
| Ghost-only | Only ghosts pass           |

---

## 🧲 Movement Physics

* Acceleration + deceleration (simple)
* Friction based on terrain

### Example:

* Road → fast
* Mud → slow
* Blight → unpredictable slow + jitter

---

## 🌊 Water Physics

* Entering water:

  * slows movement
  * stamina drain
* Deep water:

  * requires ability (or drowning risk)

---

## 🌧️ Weather Physics

* Rain → slippery ground
* Storm → movement disruption
* Wind → affects projectiles
